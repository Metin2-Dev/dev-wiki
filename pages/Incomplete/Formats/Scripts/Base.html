Script Format¶

Summary¶

The Script format is a text file that uses a declarative way of telling information, being like a dictionary that can contain Groups and Lists, considering it the JSON format of Metin2

Comments can be declared by * or // being present in the start of a line

Group¶

Groups Foo
{
  Value  "val"
  Value2  10
}

List¶

Example:

List Foo
{
  10, 20, 30, "bar"
}

Used In¶

Subsets¶

.msm (Metin2 Script Mesh) .txt (various text files, see below) .mss (Metin2 Script Sound) .prt (Object Property) .mse (Metin2 Script Effect) .msf (Metin2 Script ?) .msa (Metin2 Script animation) .sub (Subimage) .msenv (Metin2 Script Environment)
