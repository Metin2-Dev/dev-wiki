Water¶

File Extensions: .wtr

Description¶

The chunk water format specifies the water properties of an area by telling if it should be loaded and the heights of water for each X and Y coordinate

From 0 X and 0 Y to 256 X and 0 Y, following 0 X and 1 Y, 0 X and 2 Y and so on)

Composition¶

You can check the composition of the format and an example file by visualizing data formats with Kaitai and the kaitai struct below

Kaitai Struct: water.ksy
